## 1) 문자열 처리 명령어

| cmd  | description                                 |
| ---- | ------------------------------------------- |
| awk  | 입력을 주어진 분리자로 분리하여 명령을 처리 |
| diff | 파일 두개를 비교하여 다른 부분을 출력       |
| echo | 문자열이나 변수를 출력                      |
| grep | 지정한 문자열을 포함하고 있는 행을 검색     |
| sed  | 텍스트 데이터를 패턴 매칭하여 처리          |
| sort | 텍스트를 정렬                               |
